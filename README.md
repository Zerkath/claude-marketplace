# Zerkath Claude Marketplace

[Claude Code plugin marketplace](https://docs.claude.com/en/docs/claude-code/plugins).

## Install (SSH)

> [!NOTE]
> Install to Claude Code:
>
> to pin to a commit replace `main` with commit
> ```
> /plugin marketplace add git@github.com:Zerkath/claude-marketplace.git#main
> ```

## Testing Locally

> [!TIP]
> 
> `pwd` to copy the current path.
> 
> `/plugin marketplace add <path to this dir>`
